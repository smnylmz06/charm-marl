# Data

The trained models and full run logs are larger than GitHub's file size limit,
so they are not stored here. Download them from Google Drive:

https://drive.google.com/drive/folders/1F0MNTF_1hUZDUTMLBD6n7G_mJPjh8YKS?usp=drive_link

After downloading, put the files in this `data/` folder. The scripts read the
data from here, for example `data/results_all.json`.
